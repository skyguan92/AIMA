## Agent Notes

This is the first successful LLM benchmark on GB10. Peak throughput 29.0 TPS at 128-input context, with remarkably stable TPOT P95 ~34-36 ms across all input lengths. Throughput degrades gracefully to 16.8 TPS at 4096 tokens. VRAM usage is ~114 GiB (likely reporting system RAM rather than actual GPU VRAM). Compared to the 7B Qwen2.5-Coder sibling, it delivers ~2.3× higher TPS and better per-gigabyte efficiency (~5.0 TPS/GiB). All 6/6 cells passed, confirming vllm-nightly is fully functional for this model family.
